Jika a+b = 80 dan a/b = 1/4, maka nilai a - b adalah...?



Kunci Jawaban : E

PEMBAHASAN:

Apabila diketahui a + b = 80 dan a/b = 1/4, maka nilai a - b dapat dihitung dengan cara sebagai berikut:
a/b = 1/4
b = 4a .....(i)
a+b = 80 ..... (ii)

Dari persamaan diatas maka nilai a dapat dihitung:
a+b=80 è a+4a=80 è 5a=80 è a = 80/5 = 16
Apabila a = 16, maka nilai b dapat dihitung:
a+b=80 è 16+b=80 è b = 80-16 = 64

Sehingga, nilai a - b = 16 - 64 = - 48.
